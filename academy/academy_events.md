# 学院(イベント・生活)

## 学院の日課・生活スケジュール

### 基本構造
学院の1日は**3つの鐘（朝・昼・夕）**を基準に運営されている。
時間の厳密な区切りではなく、鐘の前後に柔軟な時間差がある。
貴族・庶民・教師が同じリズムで生活しており、「鐘の音で動く」という共通文化が成立している。

### 朝鐘（1日の始まり）
| 時間帯（目安） | 内容 |
|----------------|------|
| 夜明け前〜朝鐘 | 起床。宿舎内の清掃・洗顔・着替え。魔導ランプが消灯し、朝日が差す |
| 朝鐘（鳴鐘） | 学院全体の活動開始。鐘の音で全生徒が起床完了を確認 |
| 朝食（鐘の後〜1時刻） | 学院食堂にて朝食。自炊は禁止。朝は必ず"全員参加"が原則 |
| 朝礼（鐘後1.5時刻） | 教師または監督官の訓示。日直が出席確認・伝達事項 |
| 午前授業（鐘後2〜5時刻） | 読み書き・算術・魔道理論・魔術基礎などの座学中心 |

**補足:**
- 5〜7歳の初等課程では、文字や計算、簡単な魔力測定、自然観察
- 8〜10歳の中等課程では、体術・魔力制御・社会制度の基礎なども始まる
- 魔力の暴走防止のため、朝食後に軽い瞑想が義務づけられている

### 昼鐘（活動の切り替え）
| 時間帯（目安） | 内容 |
|----------------|------|
| 昼鐘（正午） | 授業終了の合図。学院全体に鐘が響く |
| 昼食（鐘の後） | 食堂または屋外で自由に昼食。貴族の子弟は軽食持参も多い |
| 自由行動（鐘後1〜2時刻） | 散歩・昼寝・図書館利用など。午睡文化がある |
| 午後授業（鐘後2〜5時刻） | 魔法実技・体術訓練・戦闘基礎など実践系授業が中心 |
| 課外実習（週数回） | 採集・調査・ギルド依頼同行などが行われる |

**補足:**
- 実技では「霊体視（霊的存在を感じ取る訓練）」も含まれるため、幻影との接触シーンを自然に組み込める
- 採集実習では、学院周辺の森林や鉱山へ小規模な班で出向く

### 夕鐘（学びの終わり）
| 時間帯（目安） | 内容 |
|----------------|------|
| 夕鐘（鳴鐘） | 授業終了と学院の閉門を知らせる。門限まで残り2時刻 |
| 夕食（鐘の後〜1時刻） | 宿舎内食堂で夕食。自由席制。平民と貴族が同席することも |
| 自由時間（鐘後1〜3時刻） | 自習・談話・外出許可者は外の市場へ。魔導灯が点灯 |
| 消灯前指導（鐘後3.5時刻） | 担当教師が巡回し、生活指導・体調確認 |
| 消灯（鐘後4時刻） | 寮の灯りが消え、静寂に包まれる。夜間外出は禁止 |

**補足:**
- 一部の研究系学徒（貴族・高位生徒）は特別許可で夜間作業を行える
- 魔導灯は学院中央制御のもとで点灯・消灯が管理される
- 眠れぬ学生たちはよく塔の鐘の音を聞きながら夜空を見上げる

### 夜間（消灯後）
- 教員・巡回警備員（魔導騎士）が寮を巡回
- 魔力探知機が稼働しており、寮の外で魔力を放出すると警告音が鳴る
- 深夜、塔の歯車が微かに鳴るのを"夜鳴き"と呼ぶ者もいる
- 「夜鳴きが聞こえる夜は、夢に過去の賢者が現れる」との言い伝えあり

## 入学イベント
### 主人公の入学
- **入学理由**: 特異スキル持ちとして学院への入学を優先される
- **家族の反応**: 家族としては「娘（主人公）が学院に行けるのは名誉」と考えており、進学を喜んで送り出す
- **本人の心境**: 主人公自身は「学院に行くのは名誉だけど、自分のスキルが何なのか不安」という葛藤を抱える

## 学院序盤イベント
### 幻影（ヘラクレス）との初邂逅
- **タイミング**: 学院の「魔力基礎」授業
- **状況**: 主人公が初めて魔力を扱う授業を受ける
- **邂逅の瞬間**: 主人公が胸の奥から魔力の流れを感じ取ると、意識の内側に声が響く「ようやく話せるか」
- **幻影の姿**: 光の中に、まだぼんやりとした幻影の姿（ヘラクレスのイメージ）が現れる

### 魔力基礎授業
- **内容**: 魔力を体内で巡らせ、外へ放出する訓練
- **主人公の反応**: 「これが魔力か…！」と感動
- **幻影の登場**: 感動の延長で幻影が現れる

## 学院中盤イベント
### 幼なじみとの再会
- **幼なじみ①（剛力）**: 冒険者として学院依頼に参加
- **幼なじみ②（冷静）**: 研究員候補として学院の学術部門から登場
- **再会の意味**: 「村の仲間」が再集結するが、それぞれ別の立場や重荷を抱えている

### 外部依頼への参加
- **依頼内容**: 学院の課題として外部依頼に同行
- **班構成**: 主人公がリーダー、幼なじみ②が参謀役
- **外部協力**: 幼なじみ①が冒険者として同行

## 学院後半イベント
### 王国からの注目
- **注目の対象**: 王国から「英雄の幻影」への注目が強まる
- **社会的影響**: 村では笑い話だったスキルが、学院や国では「英雄の再来」と認識される

### 葛藤の発生
- **主人公の立場**: 主人公は「友情」「国の思惑」「自分の理想」の間で葛藤する
- **幼なじみの立場**: 幼なじみ2人がそれぞれの立場（冒険者・研究員候補）から主人公に関わる

## 学院生活の特徴
### 孤独と成長
- **序盤**: 主人公が独自に成長し、幻影との邂逅が大きなイベントになる
- **中盤**: 幼なじみが合流し、「村の温かさ」を学院に持ち込み、孤独からの解放を描ける
- **後半**: 幻影と幼なじみの両方がそろうことで、主人公の「英雄としての道」が本格的に動き出す
