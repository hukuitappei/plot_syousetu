# 小説プロジェクト: plot_syousetu

## プロジェクト概要

このプロジェクトは、学院を舞台にしたファンタジー小説の執筆プロジェクトです。

## 構成

### エピソード1 (ep-1)
- **chapter1.md**: 第1章 - 主人公の村での生活と学院入学
- **chapter2.md**: 第2章 - 学院での初日と適応
- **chapter3.md**: 第3章 - 学院生活の始まり
- **chapter4.md**: 第4章 - 5歳の基礎授業
- **chapter5.md**: 第5章 - 6歳の新たな挑戦
- **chapter6.md**: 第6章 - 6歳秋の幻影との邂逅

### 学院設定 (academy)
学院の世界観、キャラクター、システム設定を詳細に記述したファイル群

### 会話履歴 (talk.md)
プロジェクト進行中の会話記録とアイデア

## 世界観

- **舞台**: 王立学院
- **主人公**: レイシア（特異スキル持ち）
- **時代**: 魔法が存在するファンタジー世界
- **年齢**: 5歳から始まる学院生活

## 主要キャラクター

- **レイシア**: 主人公、特異スキル「英雄の幻影」を持つ
- **アネス**: 学院の教師、母性的な存在
- **フォーネ**: ギルド員、体力指導担当
- **ディアーレ**: エルフの新入生、魔法の指導者
- **シルフィ、キリシア、ミューゼ**: 主人公の仲間たち
- **ヘラクレス**: 幻影、主人公の師となる存在

## 開発状況

- 学院設定の詳細化完了
- 主人公の成長と仲間たちとの関係構築

## ライセンス
